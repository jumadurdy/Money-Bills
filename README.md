#  Money Bills

Kişisel gider ve bütçe yönetimi uygulaması.

##  Nedir?

Money Bills, gelir ve giderlerinizi takip etmenize, 
bütçe planlamanıza ve finansal hedeflerinize 
ulaşmanıza yardımcı olur.

##  Özellikler

-  Gider ve gelir kategorileri
-  Aylık raporlar
-  Grafik analiz
-  Bütçe planlama
-  Bütçe uyarıları

##  Teknolojiler

- **Frontend:** Vue.js 3, Quasar Framework
- **State:** Pinia
- **Styling:** Tailwind CSS

##  Proje Yapısı
```
money-bills/
├── src/
│   ├── components/    # Vue componenti
│   ├── pages/         # Sayfalar
│   ├── stores/        # State management
│   ├── services/      # API çağrıları
│   └── assets/        # Resimler
├── public/            # Statik dosyalar
└── README.md          # Bu dosya
```

##  Kurulum

### Gereksinimler
- Node.js 16+
- npm veya yarn

### Adımlar
```bash
# 1. Repository'yi klonla
git clone https://github.com/jumadardy/Money-Bills.git

# 2. Klasöre gir
cd Money-Bills

# 3. Paketleri yükle
npm install

# 4. Uygulamayı başlat
npm run dev
```



##  Nasıl Kullanılır?

### Gider Eklemek
1. Dashboard'da "+" butonu tıklayın
2. Kategori seçin
3. Tutarı girin
4. "Kaydet" tıklayın

### Raporları Görüntülemek
"Raporlar" sekmesine gidin ve grafikler görün.

### Bütçe Planlama
"Bütçe" menüsünde aylık planınızı yapın.
